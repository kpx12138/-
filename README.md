# 建筑科技行业研究
###### 更新时间：2018-09-17
基于顺鑫建科集团最新战略调整和2018年重点工作，本报告主要对装配式建筑行业进行研究，并结合建科集团的优劣势，给出相关建议。

![image](https://github.com/kpx12138/sxjk/blob/master/ASaT.jpg)
## 摘要
本报告一共分三章，第一章介绍建科集团的战略目标和优势劣势，第二章研究装配式建筑行业，第三章根据前两章结论给出建科集团发展装配式建筑的意见。

核心观点：
- 利于发展的因素一直存在，制约发展的因素在逐渐消除，装配式建筑行业已经到达质变时刻，爆发式增长的时间窗口可能就在未来几年打开。
- 按照政策要求，行业推进顺利的话，以现有的装配式建筑平均价格计算，2025年的市场规模将达到4.7万亿元。
- **装配式建筑虽受到政策鼓励，但仍然缺少顶层设计和配套政策，建科集团应充分重视其中风险，认真研究相关政策。**
- **装配式建筑还处于行业早期，由于规模、成本、技术、人才等方面问题，导致项目较少、市场较小，还尚未形成市场自身的良性循环，建科集团应充分调研、谨慎论证、切实把握好投资节奏。**
- **装配式建筑各市场主体之间信息共享和协同作业程度低，严重制约着生产效率与产品质量，建科集团应充分重视其中可能存在的建筑质量风险，对项目建设高标准严要求。**
- **装配式建筑行业的各市场主体均可能成为行业主导者，建科集团应沿产业链寻找并购标的，放眼技术研发、配件制作、运输、装配等环节，不应只局限于施工单位。**

## 第一章、建科集团
### 第一节、建科集团的战略目标
根据调整后的战略规划，在“四五”期间，建科集团的战略目标是完成年营业收入“50亿” 和 “上市”两大目标，并形成可持续发展的核心竞争力；商业模式是以科技为核心、金融为工具，促进企业转型升级，聚焦乡村振兴战略，通过产业运营的方式，带动各业务单位协同发展，提升业务规模；主营业务布局是生态建设、科技建造、金融投资、产业运营；企业定位是新型城镇化建设综合服务商。

### 第二节、建科集团的优势劣势
根据内外部分析，我们可以得到，建科集团在目前发展状态下的优势和劣势：
#### 建科集团的发展优势
- 农业品牌积淀
> 顺鑫农业是北京市第一家农业上市企业，拥有20多年的农业品牌积淀，可借助“顺鑫”涉农品牌形象，整合农业产业资源。
- 国企上市背景
> 顺鑫作为顺义区最大的区属国有企业、上市公司，可以凭借其国企上市公司背景在京津冀区域及重点城市获取优质资源。
- 资金筹集优势
> 建科集团可借助自身金融工具的杠杆优势，依托控股集团平台，进一步提升自身融资及资金筹措能力，在资金层面保障项目落地。
#### 建科集团的发展劣势
- 核心能力不足、欠缺新业务案例
> 建科集团虽然具备较为完整的产业链，但是各成员企业核心能力不足，未形成整体协同，在新业务领域缺乏典型案例。
- 专业人才不足、欠缺创新机制和科技投入
> 建科集团在新业务领域缺乏大量专业性人才，对于科技研究方面投入不足，缺乏创新机制。
- 资金实力不足、难以采用资本运营方式
> 建科集团自有资金短缺、负债率高、盈利能力不足，很难实现通过资本运营的方式使企业快速实现“百亿”规模。
## 第二章、装配式建筑行业研究
装配式建筑是指建筑的结构系统、外围护系统、设备与管线系统、内装系统的主要部分采用预制部品部件集成的建筑，其最显著特征在于预制性，采用“标准化设计-工厂化生产-装配式施工”的模式，以预制构件在施工工地现场装配。

装配式建筑是我国建筑工业化的重要载体，相对于传统建筑模式，具有“环保、高效、高质”的特征。
### 第一节、装配式建筑的发展阶段
装配式建筑的概念探索，可以追溯至1950年代。从建筑行业设计标准化、生产工厂化、施工机械化，到住宅产业现代化、建筑产业现代化，最终形成了装配式建筑的概念。70年间，装配式建筑大致历经四个发展阶段：
- 第一，起步阶段：20世纪50年代。
建国初期，大量兴建厂房且建筑要求不高，具有标准化特征。在此背景之下，“一五计划”出台，强调学习苏联经验，支持装配式建筑。但受制于机械设备不完善，民用建筑应用较少。
- 第二，持续发展：20世纪60年代后。
正规构件厂开始出现（北京为主），机械设备也自国外引入。一方面，预制混凝土结构开始推广于多层民用住宅；另一方面，现浇+预制混凝土钢筋结构开始应用于高层住宅及公共建筑。
- 第三，步入低潮：20世纪80年代后。
受制于机械化水平低，装配式成本偏高，结构抗震性较差，人力成本较低等综合原因，原有已兴起的（半）预制结构建筑逐步被现浇结构取代。
- 第四，广受鼓励：2016年之后。
随着环保任务紧迫、钢铁产能过剩、人口红利消失等多种因素叠加，预制式结构建筑迎来新的发展契机。2016年，装配式建筑被写入政府工作报告，后由国务院出台文件提出，力争用10年左右时间，使装配式建筑占新建建筑的比例达到30%。2017年，住建部再次提出到2020年，全国装配式建筑占新建建筑的比例达到15%以上。地方政府也纷纷出台相关细则，鼓励装配式建筑发展。
### 第二节、装配式建筑的技术分类
装配式建筑按照结构材料可分为三大类：装配式钢结构建筑、装配式混凝土建筑和装配式木结构建筑。

> 装配式钢结构建筑:工艺模式是，在工厂采用切割后焊接的方法，在工地施工时一般采用螺栓、铆钉或者焊接的组装方法，形成能够承受荷载的钢结构；技术特点是，强度高、重量轻；应用范围是，单层厂房、大跨度公共建筑、高层和超高层的酒店和写字楼，住宅尚未广泛推广。

> 装配式混凝土建筑:工艺模式是，施工现场浇筑和工厂预制均可。预制类需在工厂制成不同的构件单元，再到工地进行组装，施工构件之间的连接可采用类似于钢结构的机械连接，也可以采用现浇钢筋混凝土组装成整体，还可以采用焊接或后张预应力的方法形成受力的结构；技术特点是，耐久性好、造价便宜；应用范围是，各类工业与民用建筑（较为主流）。

> 装配式木结构建筑:工艺模式是，一般在工厂加工成不同的构件或部品，在现场通过螺栓和螺钉连接；技术特点是，比较轻、施工简便；应用范围是，美国、加拿大、日本等国应用较多，但我国木材资源不够丰富，实际应用较少。

目前在我国，装配式建筑主要采用混凝土和钢结构两类，木结构使用的很少。其中，混凝土结构广泛应用于各类工业与民用建筑，钢结构多应用于公共建筑、厂房、桥梁等，而较少应用于住宅建筑。

装配式混凝土建筑，主要组成部分是预制混凝土构件（PC构件），当前国内建筑预制率多为20%-60%之间，预制率每提高10%，平均每单位建安成本上升约60-70元。

装配式钢结构建筑，造价相对装配式混凝土建筑要高很多，其高层住宅单位建安造价约每平米2360元，比PC率40%的装配式混凝土结构高出约300元。
### 第三节、装配式建筑行业即将质变
以历史经验来看，装配式建筑的发展，受益于“环保、高效、高质”的特征及不断扩张的建筑需求，受制于较高的生产成本和施工难度。最近几年，装配式建筑行业发展缓慢，除了高成本的原因，缺乏顶层设计、配套政策和行业标准也是重要因素。

- **通过宏观政策和产品成本的分析，我们可以看到，利于发展的因素一直存在，制约发展的因素在逐渐削弱甚至消除，由此可见，装配式建筑行业已经到达质变时刻，爆发式增长的时间窗口可能就在未来几年打开。**

> 从宏观政策角度来看，国家层面到各级省市地区，装配式建筑推广与补贴并进，政策推进力度较大，全行业有望乘政策春风，完善规范化产业化体系。

> 从产业成本角度来看，装配式建筑成本平均比传统施工高出20-25%左右，但差距正在逐步缩小。一方面，人口红利消失劳务成本走高，使得依赖现场人员的传统施工方式，正在逐渐丧失其成本优势；另一方面，行业内积极推进技术研发，增加技术人员配比，有望进一步控制装配式结构成本。
### 第四节、装配式建筑的市场规模
目前，全国31省份已出台装配式建筑发展规划（其中26个省份已出台相应补贴政策）。据估测，到2020年，约8个省份的装配式建筑面积占当期新开工面积的30%，约5个省份可达20%，预计其余17个省份该比例不低于15%。

如果推进顺利，装配式建筑市场规模将超万亿。前瞻产业研究院发布的《2018-2023年中国装配式建筑行业市场前瞻与投资规划深度分析报告》预计，按照上述方案对“2020年装配式建筑占新建建筑面积比例达15%以上”的要求测算，2020年装配式建筑面积有望超过80000万平方米，以每平米2500元测算，市场规模将超过2万亿元。如果按照政策的普遍要求，未来十年内装配式建筑将占新建建筑面积30%的比例，新增房地产需求较多的地区也是装配式建筑推进力度较多的地区，预测新增住宅面积每年提高3%，以10年为周期，以现有的装配式建筑平均价格计算，2025年的市场规模将达到4.7万亿元。

对于尚不成熟的装配式建筑行业来说，30%的目标有一定压力。行业会伴随着痛苦前行，一旦完成阶段使命，全行业发展将迎来一次飞跃。
## 第三章、综合建议
- 第一，装配式建筑虽然广受鼓励，但仍然缺少顶层设计和配套政策，建科集团应充分重视其中风险，认真研究相关政策，特别是北京市、雄安新区、京津冀周边的具体产业政策，做到“摸清规律、紧跟政策、科学规划、长远发展”。
- 第二，装配式建筑的发展窗口即将打开，但现阶段全行业还处于摸索发展的早期，由于规模、成本、技术、人才等方面问题，导致项目较少、市场较小，还尚未形成市场自身的良性循环。建科集团要尽快调研、多做试点，充分重视现金流变化、把握投资节奏，做到“新试点后推广、先慢后快”，前期要谨慎论证，后期要加快投资。
- 第三，装配式建筑现阶段的社会化程度较低、专业化分工尚未形成，这导致设计单位、施工单位、开发商、预制构件生产商之间信息共享和协同作业程度低，严重制约着装配式建筑生产效率与产品质量。建科集团应充分重视其中可能存在的建筑质量风险，高标准严要求，在国标发布之前，一切以“各细分行业最高标准”执行，并且要关注整个系统的协同性和安全性。
- 第四，由于技术标准和政策细则的空白，行业壁垒尚未形成，装配式建筑行业的各市场主体各具优势劣势，均可能成为行业主导者。建科集团在依靠自身发展的同时，要密切注视行业发展动态，沿产业链寻找并购标的，放眼技术研发、配件制作、运输、装配等环节，不应只局限于施工单位。
